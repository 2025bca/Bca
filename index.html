About the BCA Course

What is BCA?

BCA stands for Bachelor of Computer Applications. It is a 3-year undergraduate degree in computer science and IT.

Course Details

  • Semester 1: Computer Fundamentals, Programming in C
  • Semester 2: Data Structures, DBMS
  • Semester 3: Operating System, Web Technologies
  • Semester 4: Java Programming, Software Engineering
  • Semester 5: Python, Mobile App Development
  • Semester 6: Project Work and Industrial Training
